# Mi-CV
Mi primera página en HTML y CSS

Página web estática que contiene mi experiencia laboral. Aún está en construcción. 

Estructura del proyecto: 
- index.html #página principal
- styles.css #Estilos de la página
- README.md # Este archivo (README)


Cómo usarlo en local: 

No necesitas instalar nada. Sigue los siguientes pasos: 
1.- Clona o descarga el repositorio
    git clone https://github.com/vmelero13/Mi-CV.git

O descarga el ZIP desde GitHub y descomprímelo

2.- Abre la carpeta del proyecto: 
    cd index.html

3.- Abre el archivo HTML en tu navegador 
    Puedes hacerlo de dos formas: 
        - Haz doble click en index.html desde el explorador de archivos.
        - O desde la terminal
            #en Windows
            start index.html
